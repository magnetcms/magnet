# @magnet/plugin-seo

## 0.1.0

### Minor Changes

- Initial public release of Magnet CMS packages

  - @magnet/common: Shared types, decorators, and utilities
  - @magnet/utils: Utility functions
  - @magnet/core: Core NestJS module with Admin, Auth, Content, Database modules
  - @magnet/ui: UI component library built with Radix UI and Tailwind
  - @magnet/admin: React admin dashboard
  - @magnet/adapter-mongoose: Mongoose database adapter
  - @magnet/plugin-content-builder: Visual schema builder and code generator
  - @magnet/plugin-seo: SEO plugin
